Documentation

API reference, crate documentation, and integration guides for the Pyana authorization system.

Crate reference

Pyana is a 22-crate Rust workspace. When published, each crate will have full rustdoc on docs.rs.

pyana-sdk

High-level SDK: AgentWallet, AgentRuntime, SiloClient. The entry point for most integrations.

circuit

BabyBear STARK prover/verifier. AIR constraints for fold, derivation, and Merkle membership.

morpheus

Adaptive BFT consensus (Lewis-Pye & Shapiro). View-change, block finalization, Byzantine fault tolerance.

token

AuthToken trait implementations: Macaroon (HMAC-based) and Biscuit (Ed25519 + Datalog).

trace

Datalog evaluator with derivation traces. Policy evaluation that produces provable witnesses.

commit

4-ary Merkle trees with dual hash paths (BLAKE3 fast, Poseidon2 ZK). Fold delta tracking.
