Quick Start

Getting started

1

Clone and build the workspace:

git clone https://github.com/emberian/pyana
cd pyana
cargo build
2

Run the test suite:

cargo test
3

Run the end-to-end demo:

cargo run -p pyana-demo-agent
4

Build and load the browser extension:

cd extension && ./build.sh
# Load extension/manifest.json as unpacked extension in Chrome/Firefox
5

Try the browser wallet API:

// Check connection
await pyana.isConnected();

// Broadcast an intent
await pyana.postIntent({
  actions: [{ action: "read", resource: "documents/*" }],
  constraints: [{ Service: "storage" }]
});

// Listen for matches
pyana.onMatch(match => console.log("Found:", match));

API reference

SDK (pyana-sdk)

AgentWallet -- key management, minting, attenuation AgentRuntime -- turn execution, cell management SiloClient -- federation connection, root sync

Node HTTP/WS API

POST /verify -- submit proof for verification GET /roots -- attested federation roots WS /intents -- intent stream subscription

Browser Extension (window.pyana)

pyana.authorize(request) -- request capability proof pyana.postIntent(intent) -- broadcast capability need pyana.offerCapability(offer) -- respond to intent pyana.provision(token) -- import token into wallet

Key Crates

circuit -- STARK prover/verifier, AIR constraints token -- Macaroon + Biscuit token formats trace -- Datalog evaluator with derivation traces commit -- Merkle trees, fold deltas morpheus -- Adaptive BFT consensus intent -- Distributed intent engine
