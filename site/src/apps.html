Applications

Eleven production applications built on the Pyana runtime. Each one exercises a different slice of the stack — from private DeFi to federated compute markets. All generate real STARK proofs. All are runnable from the repo.

Run the examples

Every app has runnable examples. Here are a few to get started: 

# Stablecoin CDP lifecycle
cargo run -p pyana-stablecoin --example demo

# AMM swap with STARK proof
cargo run -p pyana-amm --example swap_demo

# Identity selective disclosure
cargo run -p pyana-identity --example kyc_demo

# Orderbook trading
cargo run -p pyana-orderbook --example trading_demo

# Full gallery auction
cargo run -p pyana-gallery --example devnet_gallery

See apps/*/examples/ and demo-agent/examples/ for the full catalog. The Discord bot (apps/discord-bot) also exposes 30+ slash commands for interacting with the devnet.
