Chain Explorer

Browse federation state: blocks, turns, receipts, cells, capabilities, proofs, intents, federation membership, notes, and apps. Connects to any dregg-node JSON-RPC endpoint and renders live state with embedded visualizers.

  • Block, turn, and receipt drill-down
  • Cell and capability inspection
  • Intent and queue views with embedded visualizers
  • Configurable node connection (defaults to localhost:8420)

Browser Extension Cipherclerk

With the dregg cipherclerk extension installed, web pages request capability-based authorization via window.dregg. The cipherclerk evaluates Datalog policies and generates STARK proofs locally.

  • Provision a demo token into the cipherclerk
  • Request authorization at each privacy level
  • See exactly what the verifier learns
  • All crypto runs locally in WASM

CLI Quickstart

Run the full authorization pipeline locally. The demo and demo-agent crates demonstrate minting, attenuation, proving, and multi-agent cross-silo authorization. 

git clone https://github.com/emberian/dregg
cd dregg
cargo build --release -p demo -p demo-agent

# Single-agent flow
cargo run --release -p demo

# Multi-agent cross-silo
cargo run --release -p demo-agent
