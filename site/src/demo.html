Browser Extension Wallet

With the Pyana wallet extension installed, web pages request capability-based authorization via window.pyana. The wallet evaluates Datalog policies and generates STARK proofs locally.

  • Provision a demo token into the wallet

  • See exactly what the verifier learns
  • All crypto runs locally in WASM

CLI Quickstart

Run the full authorization pipeline locally. The demo and demo-agent crates demonstrate minting, attenuation, proving, and multi-agent cross-silo authorization. 

git clone https://github.com/emberian/pyana
cd pyana
cargo build --release -p demo -p demo-agent

# Single-agent flow
cargo run --release -p demo

# Multi-agent cross-silo
cargo run --release -p demo-agent
