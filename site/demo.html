Interactive demo

Generate and verify STARK authorization proofs directly in the browser via WebAssembly. Demonstrate capability attenuation, ZK presentation, and offline verification.

WASM proof engine

The interactive playground is live. It compiles Pyana's circuit, token, commit, and trace crates to WebAssembly and runs everything client-side.

Launch Interactive Playground →

  • ▸ Build 4-ary BLAKE3 Merkle trees with membership/non-membership proofs

CLI demo

The demo and demo-agent crates provide a command-line demonstration of the full authorization flow:

# Clone and build
git clone https://github.com/emberian/pyana
cd pyana
cargo build --release -p demo -p demo-agent

# Run the demo (mints, attenuates, proves, verifies)
cargo run --release -p demo

# Run the multi-agent demo (two agents, cross-silo authorization)
cargo run --release -p demo-agent
