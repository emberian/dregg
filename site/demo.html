CLI Quickstart

Run the full authorization pipeline locally. The demo and demo-agent crates demonstrate minting, attenuation, proving, and multi-agent cross-silo authorization. Application examples exercise the full DeFi and identity stacks. 

git clone https://github.com/emberian/pyana
cd pyana
cargo build --release -p demo -p demo-agent

# Single-agent flow
cargo run --release -p demo

# Multi-agent cross-silo
cargo run --release -p demo-agent

# Application examples
cargo run --release -p demo-agent --example base_anonymous_credential
cargo run --release -p demo-agent --example base_private_transfer
cargo run --release -p demo-agent --example intent_lifecycle
