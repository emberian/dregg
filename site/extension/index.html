Install the dregg Cipherclerk Extension

The dregg Cipherclerk is a Chrome extension that manages your capabilities, evaluates Datalog policies, and generates STARK proofs — all locally in WebAssembly. No data leaves your browser.

Download the extension as a zip file and install it manually in Chrome or any Chromium-based browser.

Download dregg-cipherclerk.zip

Installation Steps

1

Download the zip using the button above. Save it anywhere convenient.

2

Unzip to a folder. Extract the zip contents to a permanent location (e.g., ~/dregg-cipherclerk/). Do not delete this folder after installing — Chrome reads from it.

3

Open the extensions page. Navigate to chrome://extensions in your browser address bar.

4

Enable Developer Mode. Toggle the "Developer mode" switch in the top-right corner of the extensions page.

5

Click "Load unpacked". A button will appear in the top-left after enabling Developer Mode. Click it.

6

Select the unzipped folder. Navigate to the folder where you extracted the zip and select it. The extension will load immediately.

Manual installation only. Chrome Web Store distribution is not yet available. The extension is fully functional including WASM-powered STARK proof generation — it just requires loading the unpacked extension as described above.

Test the Extension

Once installed, try the auth demo to see the cipherclerk in action — provision a demo token, request authorization at different privacy levels, and verify STARK proofs locally.
