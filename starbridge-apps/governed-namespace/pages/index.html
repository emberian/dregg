dregg Governed Namespace

Governance-bound atomic route table swaps on a sovereign cell. Composes dregg-dfa's GovernedRouter with dregg-cell's slot caveats (MonotonicSequence on version, Immutable on committee + threshold, SenderAuthorized against the committee root) and an Authorization::Custom threshold-signature carrier under WitnessedPredicate { kind: Custom { vk_hash: GOVERNANCE_VK } }. The executor enforces every constraint on every turn; there is no operator-side governance loop.

DFA + governance crypto + AIR-attestable accept/reject: the table-swap pattern from DFA-RATIONALIZATION-DESIGN.md §2.2, lifted onto the cell substrate. See README.md for the side-by-side comparison with the legacy apps/governed-namespace HTTP service.
